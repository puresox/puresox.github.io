  • 下图展示了最基础的用法

二、添加文献#

1. 通过 PDF 添加文献条目#

如果你已经有了一个文献的 PDF，那么可以通过将该 PDF 拖入 zotero 的方式添加一个条目。

2. 通过浏览器添加文献条目#

除了通过 PDF 添加文献条目外，最主要且方便的添加方式是通过浏览器插件添加。要让插件能添加文献到 zotero，先要让插件能够识别网页。这就用到了translators功能。软件内置了许多常用文献网站，但是还不够多。

Zotero automatically detects library records, news items, journal articles, and other objects you might like to save in your collection. Zotero uses so-called “translators” to detect and import data from websites. There are currently more than 500 different translators, facilitating data import from countless sites.

  • 官方 translators、中文社区 translators

官方的 translators 是软件内置的，我们需要额外添加中文社区的 translators，根据使用教程进行配置。

3. 其他方式#

点击菜单栏的加号创建一个空白的文献条目。 通过标识符添加条目。

七、插件#

delitemwithatt#

  • 下载地址：官网
  • 描述：删除条目或分类的同时将链接的附件也一块删除。将所选条目语言字段设置为 en。

zotero-shortdoi#

  • 下载地址：官网
  • 描述：检索和验证必须和 shortDOIs。

Jasminum#

  • 下载地址：官网
  • 描述：用于识别中文元数据。

zotero-storage-scanner#

  • 下载地址：官网
  • 描述：删除参考书目的损坏和重复附件链接。

Zotero Update IFs#

  • 下载地址：官网
  • 描述：从唯问更新影响因子Update IFs from Justscience，其他一系列工具，详见Readme

zotfile#

  • 下载地址：官网
  • 描述：Zotfile 是一个 Zotero 插件，可以管理你的附件:自动重命名，移动，并附加 PDF 文件(或其他文件)到 Zotero 项目，从你的 Zotero 库同步 PDF 文件到你的(移动)PDF 阅读器(如 iPad, Android 平板电脑等)，并从 PDF 文件提取注释。

Zutilo#

  • 下载地址：官网
  • 描述：提供一些额外的编辑功能。
