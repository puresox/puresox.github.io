一、生产工具#

1. Office 相关#

https://www.423down.com/8890.html 找了个企业版激活的WPS，没广告，速度感觉比Microsoft Office（MO）快。尤其是对于几十兆的大文件（比如我的大论文），全选之后点右键，MO要等十几秒才会出右键菜单，WPS立刻就能出来。 阻止我换WPS的原因之一是他和MO兼容的问题，我用MO写的大论文用WPS打开有的地方格式会不对（比如我的摘要word打开正好一页，wps打开就一页多一行，很离谱）。原因之二是WPS在ctrl+s一个大文件之后就无法进行撤销动作。

Office Tool Plus#

Office Tool Plus 是一个用于部署、激活 Office、Visio、Project 的小工具。借助本工具，你可以快速地完成各项 Office 部署工作！

  • 下载地址：官网
  • 描述：用来安装和管理 Office。
  • 备注：Office 激活请看我其他文章，使用HEU_KMS_Activator可以激活Office 365。

MathType#

MathType是一款强大的数学公式编辑器

  • 下载地址：白嫖版
  • 描述：亲测有效。
  • 备注：配合GrindEQ实现新版公式转换为旧版公式（word 2003）。

2. 同步工具#

我工作需要用到两台电脑，一台笔记本，上面会有一些私密的数据，一台是公家的台式机，专门处理公家的事务。由此产生了两个需求，一个备份，将数据备份到云盘或者其他硬盘，我用的是 FileGee 实现这个功能；另一个是在两台电脑之间同步数据，我用的是 PP 直连实现这个功能。此外，对于代码，我用 git。

FileGee#

集文件备份、同步、加密、分割、合并于一身，实现高效化办公。

  • 下载地址：官网
  • 描述：可实现 FTP、网络共享、特定网盘及本机各种存储器之间的备份与同步。
  • 备注：我用来将文件同步或备份到网盘和 U 盘。

PP 直连#

点到点直连，高速、私密的文件传输、实时共享、远程控制

  • 下载地址：官网
  • 描述：点对点传输，无需中间服务器。
  • 备注：可以直接在浏览器中使用，很方便。Syncthing 不好使了，传输太慢。

3. 效率工具#

Everything#

Everything. Locate files and folders by name instantly. Everything. Small installation file. Clean and simple user interface. Quick file indexing. Quick searching.

  • 下载地址：官网
  • 描述：可以非常快速地搜索文件。
  • 备注：替代 Windows 自带的搜索。

Maye#

Maye 是一款体积小巧、简单易用的快速启动工具。不要看体积小巧，它的功能还是非常多样化的，比如：多文件拖拽添加启动、快捷键呼出、自动多列显示、lnk 文件解析等等。软件没有繁杂的功能，操作中也不会产生任何垃圾文件，绿色无污染，只专注于文件的快速启动，是用户提升 Windows 操作体验的好帮手。

  • 下载地址：官网
  • 描述：Windows11 的动态磁贴没有了，所以使用这款快捷启动工具。
  • 备注：如果你不习惯 win11 的 win+S 启动逻辑，可以试试用这个替代 Windows10 的动态磁贴。

PDF 补丁丁#

PDF 补丁丁是一个多功能的 PDF 文档工具箱，在 2009 年开始，我开始了该程序的开发，到现在也已经有十二年了。它致力于解除 PDF 文档的烦恼，带有一个强大的 PDF 书签编辑器（可自动生成书签），有超快的从 PDF 文档里无损提取图片的能力，又可以合并图片和 PDF 文档，统一页面尺寸，清除文档的打印和复制限制，这些功能都是这些年来一直为用户所喜爱的。对于高端开发者，又提供了文档结构探查器等一系列功能。

  • 下载地址：官网
  • 描述：PDF 文档工具箱。
  • 备注：一些保密文件用在线服务不放心，需要一个离弦的开源工具箱。
