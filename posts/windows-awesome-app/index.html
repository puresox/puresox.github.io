一、生产工具

1. Office 相关

Office Tool Plus

Office Tool Plus 是一个用于部署、激活 Office、Visio、Project 的小工具。借助本工具，你可以快速地完成各项 Office 部署工作！

  • 下载地址：官网
  • 描述：用来安装和管理 Office。
  • 备注：Office 激活请看我其他文章。

2. 同步工具

Syncthing

Syncthing is a continuous file synchronization program. It synchronizes files between two or more computers in real time, safely protected from prying eyes.

  • 下载地址：官网
  • 描述：用来在多台设备上同步文件。
  • 备注：下载可能需要使用科学上网，但是使用不需要。

FileGee

集文件备份、同步、加密、分割、合并于一身，实现高效化办公。

  • 下载地址：官网
  • 描述：可实现 FTP、网络共享、特定网盘及本机各种存储器之间的备份与同步。
  • 备注：我用来将文件同步或备份到网盘和 U 盘。

3. 效率工具

Everything

Everything. Locate files and folders by name instantly. Everything. Small installation file. Clean and simple user interface. Quick file indexing. Quick searching.

  • 下载地址：官网
  • 描述：可以非常快速地搜索文件。
  • 备注：替代 Windows 自带的搜索。

Maye

Maye 是一款体积小巧、简单易用的快速启动工具。不要看体积小巧，它的功能还是非常多样化的，比如：多文件拖拽添加启动、快捷键呼出、自动多列显示、lnk 文件解析等等。软件没有繁杂的功能，操作中也不会产生任何垃圾文件，绿色无污染，只专注于文件的快速启动，是用户提升 Windows 操作体验的好帮手。

  • 下载地址：官网
  • 描述：Windows11 的动态磁贴没有了，所以使用这款快捷启动工具。
  • 备注：替代 Windows10 的动态磁贴。

Quicker

Quicker,Windows 效率神器。为您的常用 Windows 操作建立捷径，更快、更简单的完成工作，让效率触手可及。

  • 下载地址：官网
  • 描述：真·效率神器，谁用谁知道。有很多使用插件可以选，搭配沙拉查词可以实现全局划词翻译。
  • 备注：无。

4. 学习

Zotero

Zotero is a free, easy-to-use tool to help you collect, organize, cite, and share research.

  • 下载地址：官网
  • 描述：管理文献。
  • 备注：自动下载文献、云盘同步请看我的。

Logseq

Logseq is a joyful, open-source outliner that works on top of local plain-text Markdown and Org-mode files. Use it to write, organize and share your thoughts, keep your to-do list, and build your own digital garden.

  • 下载地址：官网
  • 描述：一个知识库，用来管理笔记，文档等知识。
  • 备注：双向链接。

MATLAB

MATLAB is a programming and numeric computing platform used by millions of engineers and scientists to analyze data, develop algorithms, and create models.

  • 下载地址：破解版
  • 描述：科研要用。
  • 备注：破解版网上很多，貌似最初是俄罗斯人破解的，然后传过来的。

5. 远程终端

WindTerm

A Quicker and better SSH/Telnet/Serial/Shell/Sftp client for DevOps.

  • 下载地址：官网
  • 描述：方便，功能多。
  • 备注：之前用的 MobaXterm，现在换成 WindTerm。

Windows Terminal

Windows 终端程序是一款新式、快速、高效、强大且高效的终端应用程序，适用于命令行工具和命令提示符， Powershell 和 WSL 等 Shel 用户。主要功能包括多个选项卡、窗格、 Unicode、和 UTF-8 字符支持，GPU 加速文本渲染引擎以及自定义主题、样式和配置。

  • 下载地址：官网
  • 描述：支持美化，支持多种 shell，我用的 PowerShell7 替代 cmd 和 Windows 自带的老版 PowerShell。
  • 备注：设置以及美化请看我其他文章。

远程桌面

Microsoft 远程桌面客户端使你可以使用和控制远程电脑。 对于远程桌面客户端，可以使用远程电脑完成所有操作，就像使用物理电脑一样，例如：

  • 使用安装在远程电脑上的应用。
  • 访问远程电脑上的文件和网络资源。
  • 在关闭客户端时使应用保持打开状态。
  • 下载地址：官网
  • 描述：微软出品的远程桌面，简洁且强大，不需要远程连接的电脑有显示器，可以传输文件。
  • 备注：你可以理解为把你当前的显示器作为一个远程无线显示器，而不是像替他远程桌面一样只是把桌面图像传过来。

Visual Studio Code

VS Code is my most used and favorite editor. I love being able to customize the editor - changing the layout, the icons, fonts and color scheme is so easy!

  • 下载地址：官网
  • 描述：Visual Studio 是史上最强 IDE，Visual Studio Code 是史上最强编辑器。
  • 备注：设置及扩展推荐请看我其他文章。

Hugo

Hugo is one of the most popular open-source static site generators. With its amazing speed and flexibility, Hugo makes building websites fun again.

  • 下载地址：官网

四、多媒体

1. 图片

关于图片浏览和编辑的软件分成了两派：一派是“全”，把浏览和编辑集中在一个软件中，比如 FastStone Image Viewer；另一派是“专”，浏览归浏览，讲究的是快速，稳定，轻量，编辑归编辑，讲究的是专业。针对第二派，对于轻量需求用户，可以用 Windows 系统自带的照片和画图用于图片的浏览和编辑；对于重度需求用户，编辑就要用到 PS 等专业软件了。

FastStone Image Viewer

FastStone Image Viewer is a fast, stable, user-friendly image browser, converter and editor. It has a nice array of features that include image viewing, management, comparison, red-eye removal, emailing, resizing, cropping, retouching and color adjustments.

  • 下载地址：官网
  • 描述：图片浏览，快，功能多。
  • 备注：无。

🚀A simple & beautiful tool for pictures uploading built by vue-cli-electron-builder.
